技术组件

  • Spring Cloud

    当前最流行的微服务开发框架SpringCloud，数人云提供全面支持，包括配置中心，注册中心，服务治理中心，API Gateway等多个产品都无缝对接SpringCloud体系。

  • Hawk

    数人云统一配置管理中心 Hawk，为分布式系统中的基础设施和微服务应用提供集中化的外部配置支持，实现了对服务端和客户端中环境变量，配置文件，动态属性配置的抽象映射，并支持配置的灰度下发，历史版本控制等先进功能。配置接口完全兼容 Spring Cloud Config，后台管理功能则通过 open API的形式对外开放，满足企业多样化的定制需求。

  • Nemo

    Nemo提供基于Kubernetes的容器化管理，使用自服务形式，通过一体化管理控制台，提供通用中间件的集群部署、管理和监控，也提供中间件自身特性的管理和监控。Nemo深度集成行业OSB(Open Service Broker)标准，沉淀生产环境最佳实践，标准化提升中间件的管理、性能、安全

  • Coral

    数人云微服务治理中心 Coral，为微服务体系提供注册与治理的能力，是服务调用者和服务提供者之间的桥梁和网络。Coral 完全兼容 spring cloud eureka 的接口，并能治理使用 ribbon，hystrix 等 spring cloud 组件的微服务应用，无缝接入 spring cloud 体系，提供企业级的服务治理能力。

  • Octopus

    Octopus 基于数人云DataMan OS 容器云平台，传统的定时作业程序可以无缝迁移到容器云上运行，通过取代传统的Linux Cron/Spring Batch Job/Quartz，做到分布式环境下批量作业任务的全域统一配置，统一监控，任务高可用以及分片并行。

  • API Gateway-Whale

    轻松创建，发布，维护，监控和保护API，只需在控制台轻松操作，即可暴露微服务的API给客户端应用。提供流量管理，授权和访问控制，监控以及版本控制。

  • Kubernetes

    云原生管理跨多台主机容器化应用的开源系统。

  • Mesos

    分布式操作系统内核，也是数人云 DataMan OS 容器云平台的核心组件。将整个集群抽象为一个资源池，可像用一台电脑一样管理整个集群资源。

  • Docker

    Docker 容器是数人云 DataMan OS 容器云平台的核心组件。使用数人云可轻松发布和管理 Docker 容器化应用（包括 Docker Hub 及其他第三方平台上的 Docker 镜像）。

  • Swan

    Swan 是基于 Mesos 实现的应用调度框架，帮助用户轻松发布应用，实现应用的滚动更新，并根据用户指定的策略做应用的健康检测和故障转移。
