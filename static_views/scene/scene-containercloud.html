数人云 - 下一代 DCOS

企业级容器云 - 基于 Mesos 的最成熟 Docker 生产环境

一键伸缩
满足业务爆发增长需求

数人云能够根据客户实际业务需求，动态伸缩容器实例数量。 轻松应对业务快速增长带来的高并发问题。

一键部署 —— 快速搭建容器生产环境

可一键部署 Docker 容器化应用。（支持企业原有容器化应用，及 Docker Hub 等第三方平台的 Docker 镜像）通过将应用、依赖关系、配置信息打包在一起，Docker 封装的应用拥有了极高的可移植性。这样就使得用户不需要重写代码，就可以在公有云、私有云、或混合云上部署和管理应用。

应用编排 —— 大规模微服务运行平台

定制模板化应用依赖关系，帮助客户快速部署和维护复杂的微服务架构应用。

简化运维 —— 快速实现 DevOps

打破开发和运维团队之间的障碍，提高应用开发、部署、维护效率。通过统一界面，管理应用和集群。可随时监控每一个应用，集群，主机的运行健康状态。

提供 API 接口，可与多种第三方应用，或企业自有管理系统对接整合。可统一收集应用和集群主机日志，能够快速查询和检索，帮助企业快速定位问题。让用户像用单机电脑一样管理集群和云端应用。

服务高可用 —— 自动容错，永不掉线

当您开启高可用服务后， 数人云自动为宕机服务器上运行的节点重新分配资源，保障业务不掉线，高可靠运行。这也就意味着您不用再为一两台服务器的宕机， 而经历一个不眠之夜。
