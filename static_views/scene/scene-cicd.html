问题和分析

如今互联网时代，对于每一个企业，产品快速迭代的重要性不言而喻。但是持续集成和持续交付（CI/CD）其实并没有那么容易实现，单是部署一个稳定的 Jenkins 持续集成环境，就很费时费力，更不要说持续交付过程中保证应用平滑升级，避免服务宕机。

解决方案

数人云提供企业级持续集成和持续交付解决方案，从快速搭建持续集成环境，到改进和融合企业的开发、测试及持续发布流程，助力企业提升产品发布效率，规范产品开发流程。

数人云以 Mesos 技术为核心，面向数据中心对计算资源进行统一管理和调度，同时通过 Docker 技术对应用进行统一封装和分发，实现企业应用的快速发布。

场景

数人云从以下几个方面为客户进行持续集成方案落地：

  • 版本控制：可以继承企业现有的版本控制工具，如 Github、GitLab、SVN、CVS 等主流工具。
  • 构建及测试：通过 Jenkins 实现自动构建和测试，根据企业现有技术栈及开发语言，提供构建及测试流程培训。
  • 交付：以 Docker 镜像形式进行交付，提交至镜像仓库；数人云可以提供私有部署的企业级镜像仓库。
  • 发布：可通过自动、或人工触发的方式，将应用新版本发布或更新至数人云集群，支持应用的灰度发布，保证应用平滑升级。
