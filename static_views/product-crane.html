数人云容器管理面板

国内首个基于 Docker SwarmKit 的容器管理工具「体验版」

GitHub Project

Characteristic

关于数人云容器管理面板 Crane
国内首个基于最新 Docker SwarmKit 技术的集群管理工具。它根据 Docker 的原生编排功能，采用轻量化架构，帮助开发者快速搭建 DevOps 环境，体验 Docker 的各种最新功能。

应用场景

Scenarios

Scenarios-1

快速搭建开发/测试环境，一键部署，节省运维主机资源成本。

Scenarios-2

应用隔离，极大提高数据中心资源利用率。

Scenarios-3

基于 Docker SwarmKit 实现规模集群管理，应用弹性扩缩。

功能特性

Feature

Feature-1

应用治理

应用跨机编排

保证应用内服务的自动发现，有多种应用下发模式

服务滚动更新

控制固定时间内的并行更新任务数，与上游的镜像构建对接

任务按需伸缩

对运行中的服务随时进行扩容缩容，并保证全局的资源隔离

服务全局容错

配置任务的失败重启策略，保证集群的整体稳定性

多机负载均衡

提高了服务的可用性和吞吐量，消除负载均衡单点

多种分发模式

实现多种任务分发组合，满足不同应用场景

Feature-2

镜像管控

应用目录

打造企业内部的镜像市场，一键下发公共应用，提高运维效率

私有镜像

实现用户管理与镜像管控的闭环

公有镜像

结合用户管理功能，将镜像公开给组内的人使用，实现组内资源共享

Feature-3

集群运维

主机管理

查看集群内各个主机；随时快速添加主机；一键调度；为主机添加标签等

虚拟网络

实现应用网络隔离和独立的网络规划能力，无缝对接多种 overlay 网络方案

webssh

进入容器内部进行操作，同时限制用户进入他人的容器内部，张弛有度

实时监控

从全局和底层单元多角度监控主机和服务的运行状况

实时日志

容器和服务级别的实时日志收集

Feature-4

仓库认证

镜像仓库的认证管理

用户可以管理自己多个镜像仓库的认证信息。在发布应用时，可以利用认证信息拉取相应仓库的镜像到集群中进行部署。

Feature-5

轻量化设计

安装便捷

十分钟，零基础搞定安装

插件化架构

灵活插拔存储与网络驱动，按需启停用户管理，镜像管理等功能

对接上下游

提供服务更新的 webhook 对接上游镜像构建

开放接口

提供命令行工具和 API 接口，无缝对接企业内部运维流程
